# Proyecto tipo Red Social

Proyecto personal creado con el objetivo de **aprender, practicar y consolidar conceptos reales de desarrollo web**, utilizando Next.js como framework fullstack.

La idea es simular una pequeña red social, donde los usuarios puedan registrarse, iniciar sesión y tener su propio perfil con publicaciones, aplicando patrones y herramientas usadas en proyectos reales.

## Objetivos del proyecto

- Implementar autenticación real (login y registro)
- Manejar sesiones y rutas protegidas
- Aprender el uso de JWT y cookies seguras
- Aplicar validaciones tanto en frontend como backend
- Usar librerías populares y buenas prácticas
- Mejorar la organización y mantenibilidad del código
- Trabajar la estética y la experiencia de usuario

## Tecnologías principales

- **Next.js** 
- **TypeScript**
- **Tailwind CSS**
- **Zod** – Validaciones
- **JWT** – Autenticación
- **PostgreSQL** (a integrar)
- **OAuth** (planificado)
